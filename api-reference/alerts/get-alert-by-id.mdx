---
title: Get Alert by ID
api: GET /alerts/{id}
description: Retrieve a single alert by its numeric ID
---

## Parameters

<ParamField path="id" type="integer" required>
  The unique alert ID.
</ParamField>

## Response

Returns a single [Alert object](/reference/alert-object).

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer YOUR_KEY" \
  https://polychadsbot.xyz/api/v1/alerts/2220
```

```python Python
import requests

resp = requests.get(
    "https://polychadsbot.xyz/api/v1/alerts/2220",
    headers={"Authorization": "Bearer YOUR_KEY"}
)
alert = resp.json()
print(f"Signal: {alert['signal_score']}")
```

</RequestExample>
