---
title: Get Alerts
api: GET /alerts
description: Paginated alert history with powerful filtering
---

## Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination.
</ParamField>

<ParamField query="page_size" type="integer" default="20">
  Results per page. Maximum depends on your tier (Free: 20, Pro: 50, Enterprise: 100).
</ParamField>

<ParamField query="category" type="string">
  Filter by market category. Options: `crypto`, `politics`, `world`, `tech`, `finance`, `pop_culture`, `other`.
</ParamField>

<ParamField query="side" type="string">
  Filter by trade direction. Options: `BUY`, `SELL`.
</ParamField>

<ParamField query="min_usdc" type="float">
  Minimum trade size in USDC.
</ParamField>

<ParamField query="min_signal" type="integer">
  Minimum signal score (0â€“100). Only return alerts above this threshold.
</ParamField>

## Response

Returns a paginated object containing an array of [Alert objects](/reference/alert-object).

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer YOUR_KEY" \
  "https://polychadsbot.xyz/api/v1/alerts?category=crypto&side=BUY&min_signal=60&page=1"
```

```python Python
import requests

resp = requests.get(
    "https://polychadsbot.xyz/api/v1/alerts",
    headers={"Authorization": "Bearer YOUR_KEY"},
    params={
        "category": "crypto",
        "side": "BUY",
        "min_signal": 60,
        "page": 1
    }
)
data = resp.json()
print(f"Total: {data['total']}, Page: {data['page']}")
```

</RequestExample>
