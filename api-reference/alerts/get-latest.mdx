---
title: Get Latest Alerts
api: GET /alerts/latest
description: Retrieve the most recent insider alerts
---

## Parameters

<ParamField query="limit" type="integer" default="10">
  Number of alerts to return. Range: 1â€“50.
</ParamField>

## Response

Returns an array of [Alert objects](/reference/alert-object), sorted newest first.

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer YOUR_KEY" \
  "https://polychadsbot.xyz/api/v1/alerts/latest?limit=2"
```

```python Python
import requests

resp = requests.get(
    "https://polychadsbot.xyz/api/v1/alerts/latest",
    headers={"Authorization": "Bearer YOUR_KEY"},
    params={"limit": 2}
)
alerts = resp.json()
```

```javascript JavaScript
const resp = await fetch(
  "https://polychadsbot.xyz/api/v1/alerts/latest?limit=2",
  { headers: { Authorization: `Bearer ${apiKey}` } }
);
const alerts = await resp.json();
```

</RequestExample>
