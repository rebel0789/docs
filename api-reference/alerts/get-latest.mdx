---
title: Get Latest Alerts
openapi: GET /alerts/latest
description: Most recent insider alerts
---

Returns the latest alerts, sorted newest first.

<ParamField query="limit" type="integer" default="10">
  How many to return. Max 50.
</ParamField>

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer YOUR_KEY" \
  "https://polychadsbot.xyz/api/v1/alerts/latest?limit=2"
```

```python Python
import requests

r = requests.get(
    "https://polychadsbot.xyz/api/v1/alerts/latest",
    headers={"Authorization": "Bearer YOUR_KEY"},
    params={"limit": 2}
)
for alert in r.json():
    print(f"{alert['side']} {alert['outcome']} — ${alert['usdc_value']} — score {alert['signal_score']}")
```

```javascript JavaScript
const r = await fetch(
  "https://polychadsbot.xyz/api/v1/alerts/latest?limit=2",
  { headers: { Authorization: `Bearer ${apiKey}` } }
);
const alerts = await r.json();
```
