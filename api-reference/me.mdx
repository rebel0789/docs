---
title: Get My Info
api: GET /me
description: Check your API key's tier, limits, and usage
---

## Response

Returns information about your API key including tier, rate limits, and today's request count.

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer YOUR_KEY" \
  https://polychadsbot.xyz/api/v1/me
```

</RequestExample>

<ResponseExample>

```json 200
{
  "name": "MyApp",
  "tier": "pro",
  "rate_limit": "300/min",
  "websocket": true,
  "webhooks": false,
  "realtime": true,
  "max_page_size": 50,
  "requests_today": 142
}
```

</ResponseExample>

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | Name assigned to this key |
| `tier` | string | `free`, `pro`, or `enterprise` |
| `rate_limit` | string | Your rate limit (e.g. `300/min`) |
| `websocket` | bool | Whether WebSocket access is enabled |
| `webhooks` | bool | Whether webhook delivery is enabled |
| `realtime` | bool | Whether you get real-time data (vs 5-min delay) |
| `max_page_size` | int | Max results per page for `/alerts` |
| `requests_today` | int | API requests made today |

<Tip>
Use this endpoint to check your remaining quota before hitting rate limits.
</Tip>
