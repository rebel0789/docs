---
title: Get Stats
api: GET /stats
description: System-wide alert statistics
---

<RequestExample>

```bash cURL
curl -H "Authorization: Bearer YOUR_KEY" \
  https://polychadsbot.xyz/api/v1/stats
```

</RequestExample>

<ResponseExample>

```json 200
{
  "total_alerts": 2226,
  "alerts_today": 183,
  "avg_signal_today": 59.2,
  "clusters_today": 0,
  "win_rate": 0.0,
  "total_resolved": 0,
  "total_won": 0
}
```

</ResponseExample>

| Field | Type | Description |
|-------|------|-------------|
| `total_alerts` | int | All-time alert count |
| `alerts_today` | int | Today's alerts (UTC) |
| `avg_signal_today` | float | Mean signal score today |
| `clusters_today` | int | Cluster alerts today |
| `win_rate` | float | % of resolved alerts that won |
| `total_resolved` | int | Markets that resolved |
| `total_won` | int | Trades that hit |
