## API Keys

All API requests require an API key passed in the `Authorization` header using Bearer token format.

```bash
Authorization: Bearer YOUR_API_KEY
```

<Info>
Contact us on [Telegram](https://t.me/polychads) to receive your API key and choose your tier.
</Info>

## Making Requests

<CodeGroup>

```bash cURL
curl -H "Authorization: Bearer pc_your_key_here" \
  https://polychadsbot.xyz/api/v1/me
```

```python Python
import requests

resp = requests.get(
    "https://polychadsbot.xyz/api/v1/me",
    headers={"Authorization": "Bearer pc_your_key_here"}
)
print(resp.json())
```

```javascript JavaScript
const resp = await fetch("https://polychadsbot.xyz/api/v1/me", {
  headers: { "Authorization": `Bearer ${apiKey}` }
});
const data = await resp.json();
```

</CodeGroup>

## Key Format

API keys follow the format `pc_` followed by a random string:

```
pc_Kr_Ohzog-pwKB64HmOk7uGf-Q86Zde5kpyD1F2IVDzA
```

## Verify Your Key

Use the `/me` endpoint to verify your key is working and check your tier:

```bash
curl -H "Authorization: Bearer YOUR_KEY" \
  https://polychadsbot.xyz/api/v1/me
```
