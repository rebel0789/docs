<Steps>
  <Step title="Get your API key">
    Open [@chadsapibot](https://t.me/chadsapibot) on Telegram. Tap **Get API Key**, join our alerts channel, and choose your tier. You'll get a key like `pc_Kr_Ohz...`.

    <Tip>Pro tier is free for 30 days — real-time data, WebSocket, 300 req/min.</Tip>
  </Step>

  <Step title="Verify your key">
    ```bash
    curl -H "Authorization: Bearer YOUR_KEY" \
      https://polychadsbot.xyz/api/v1/me
    ```

    You'll see your tier, rate limit, and today's usage.
  </Step>

  <Step title="Pull latest alerts">
    ```bash
    curl -H "Authorization: Bearer YOUR_KEY" \
      "https://polychadsbot.xyz/api/v1/alerts/latest?limit=5"
    ```

    Returns the 5 most recent insider alerts with signal scores, prices, and wallet data.
  </Step>

  <Step title="Check hot markets">
    ```bash
    curl -H "Authorization: Bearer YOUR_KEY" \
      "https://polychadsbot.xyz/api/v1/hot-markets?period=24h"
    ```

    See which markets have the most insider activity — ranked by alert count, signal score, and trade volume.
  </Step>

  <Step title="Go real-time (Pro)">
    ```javascript
    const ws = new WebSocket(
      "wss://polychadsbot.xyz/ws/alerts?api_key=YOUR_KEY"
    );

    ws.onmessage = (e) => {
      const msg = JSON.parse(e.data);
      if (msg.type === "alert") {
        console.log(msg.data.event_title);
        console.log(`${msg.data.side} ${msg.data.outcome} — $${msg.data.usdc_value}`);
      }
    };

    // Keep-alive every 25s
    setInterval(() => {
      if (ws.readyState === WebSocket.OPEN) ws.send("ping");
    }, 25000);
    ```
  </Step>
</Steps>
