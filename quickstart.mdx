## Step-by-Step

<Steps>
  <Step title="Get your API key">
    Contact us on [Telegram](https://t.me/polychads) to receive your API key. Choose between **Free**, **Pro**, or **Enterprise** tiers.
  </Step>

  <Step title="Verify your key works">
    ```bash
    curl -H "Authorization: Bearer YOUR_KEY" \
      https://polychadsbot.xyz/api/v1/me
    ```

    You should see your key info including tier and rate limits.
  </Step>

  <Step title="Fetch the latest alerts">
    ```bash
    curl -H "Authorization: Bearer YOUR_KEY" \
      "https://polychadsbot.xyz/api/v1/alerts/latest?limit=5"
    ```

    This returns the 5 most recent insider alerts.
  </Step>

  <Step title="Connect to WebSocket (Pro+)">
    For real-time streaming, connect via WebSocket:

    ```javascript
    const ws = new WebSocket(
      "wss://polychadsbot.xyz/ws/alerts?api_key=YOUR_KEY"
    );

    ws.onmessage = (e) => {
      const msg = JSON.parse(e.data);
      if (msg.type === "alert") {
        console.log("ðŸ”” New alert:", msg.data);
        console.log(`  ${msg.data.side} ${msg.data.outcome} @ ${msg.data.entry_price}`);
        console.log(`  $${msg.data.usdc_value} | Signal: ${msg.data.signal_score}`);
      }
    };
    ```
  </Step>
</Steps>
