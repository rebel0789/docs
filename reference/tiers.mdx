## Free Tier

The free tier gives you full REST API access with a 5-minute delay on data. Perfect for prototyping and evaluating the data quality.

<Note>
Free tier alerts are delayed by 5 minutes from real-time. All other data (stats, your key info) is live.
</Note>

## Pro Tier

Real-time data access with WebSocket streaming. Ideal for bots, dashboards, and trading applications.

**Includes:**
- Real-time alert delivery (zero delay)
- WebSocket streaming for push-based alerts
- 300 requests per minute
- Up to 50 results per page

## Enterprise Tier

Everything in Pro, plus webhook delivery and unlimited rate limits. Built for production applications that need guaranteed reliability.

**Includes:**
- Everything in Pro
- Webhook HTTP POST delivery
- Unlimited API requests
- Up to 100 results per page
- Priority support
