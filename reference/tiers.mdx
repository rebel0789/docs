## Free

Full REST API access with a 5-minute delay on alerts. Good for evaluating the data and building prototypes.

## Pro

Everything in Free, plus real-time data and WebSocket streaming. Built for bots and dashboards that need live alerts.

## Enterprise

Everything in Pro, plus webhook delivery and no rate limits. For production systems that can't afford to miss an alert.
